# Overview
Collections are a way to group games together under one label. These groups can then be assigned to roles so you can control which games are available to your users.

## Adding a Collection
There are currently two ways to add a new collection.

1. In the main navigation, go to Metadata / Collections then click the "Add Collection" button
2. When a game is selected under the main games list, click the "Add to Collection" button. Click the dropdown list in the dialog and use the input field at the bottom to name your new collection.

## Adding Games to a Collection
There are currently two ways to add a game to an existing collection:

1. When editing a game, use the "Collections" field to choose all applicable collections
2. Select the game(s) using the checkboxes in the main game list and click the "Add to Collection" button. Use the dropdown to select the applicable collection.

## Assigning a Collection to a Role
In the main navigation, go to Settings / Roles and then click edit on the role you wish to assign a collection to. A multiselect field will visible. Use this to assign as many collections as desired to the role.

:::info
The **Administrator** role will be able to see all collections of all games regardless of their assignment or restriction.
:::
